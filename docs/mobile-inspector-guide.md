# MZStay Mobile 检查人员实用指南

适用角色：检查人员 Cleaning Inspector
也适用：Cleaner Inspector（清洁 + 检查混合角色）
适用端：MZStay 移动端 App

## 1. 每天开始前先看任务

进入 App 后打开底部“任务”页。检查人员主要看检查任务，任务卡通常会显示房号、地址、入住时间、检查人员、清洁人员、旧密码、新密码、客人需求、状态等信息。

操作步骤：

1. 确认顶部日期是今天。
2. 查看是否有“待检查”任务。
3. 优先处理早入住、高优先、客人快到、已经挂钥匙的任务。
4. 点开任务前先看房号、地址、入住时间和备注。
5. 如果任务不是自己负责，先不要处理，联系负责人确认。

## 2. 前一天或当天先排检查顺序

检查人员可以在任务页使用“排序”。纯检查人员只排序检查任务；清洁 + 检查混合角色可以排序清洁任务和检查任务。

建议前一天晚上或当天开始前先排好检查路线，避免来回跑。

操作步骤：

1. 在任务页顶部选择日期。
2. 检查任务数量和房源是否正确。
3. 点击右侧“排序”。
4. 按实际检查路线依次点选任务卡。
5. 全部点完后点击“保存顺序”。
6. 当天按保存后的顺序执行。

排序原则：

1. 早入住、客人快到的房源优先。
2. 已经挂钥匙、可马上检查的房源优先。
3. 同一区域房源尽量连在一起。
4. 有客人特殊需求的房源要提前留出检查时间。
5. 如果清洁还没完成，不要只按路线排，要结合现场完成状态。

## 3. 进入“检查与补充”

在检查任务卡下方点击“检查与补充”。这是检查人员最主要的工作页面。

页面分为五块：

1. 消耗品补充
2. 清洁问题反馈
3. 房源问题反馈
4. 房间检查照片
5. 标记已完成

建议按页面顺序做，不要直接跳到完成。

## 4. 消耗品补充

进入检查页面后先处理消耗品。

如果没有待补充项：

1. 现场确认纸巾、咖啡、洗漱、厨房用品等都充足。
2. 点击“确认都充足”。
3. 页面显示“消耗品已确认充足”后再继续。

如果有待补充项：

1. 查看每个补充项名称和建议数量。
2. 已经补上的，选择“已补充”。
3. 无需补的，选择“无需补充”。
4. 选择“已补充”的项目必须拍照上传。
5. 可填写备注，例如“仓库缺货”“已补 2 个”等。
6. 点击“提交消耗品补充”。

注意：

1. “无需补充”不需要上传照片。
2. “已补充”没有照片时不能算完成。
3. 如果清洁员已经上传过补货照片，检查员仍要按现场实际确认。

## 5. 清洁问题反馈

如果发现清洁不到位，在“清洁问题反馈”里记录。

适用情况：

1. 地毯、地面、厨房、浴室没有清洁干净。
2. 床、沙发、桌面、玻璃有明显污渍。
3. 垃圾没有清走。
4. 补品摆放错误或明显缺漏。

操作步骤：

1. 展开“清洁问题反馈”。
2. 点击“拍照上传”或“相册上传”。
3. 给每张问题照片写清楚备注。
4. 如果拍错，可以删除。

备注建议写具体位置，不要只写“没弄好”。例如：“厨房水槽有油渍”“主卧地毯有毛发”。

## 6. 房源问题反馈

如果发现不是清洁问题，而是房源本身的问题，进入“房源问题反馈”提交。

适用情况：

1. 家具、电器、门锁、窗帘损坏。
2. 漏水、堵塞、异味、墙面破损。
3. 客人遗留物。
4. 设施无法正常使用。
5. 房源信息、密码、指南与现场不一致。

操作步骤：

1. 展开“房源问题反馈”。
2. 点击“进入问题反馈”。
3. 选择问题类型。
4. 写清楚位置、情况、影响。
5. 上传现场照片。
6. 提交后通知负责人查看。

## 7. 房间检查照片

检查照片是完成检查前的关键步骤。正常情况下，每个区域至少 1 张才能继续完成。

需要拍的区域：

1. 客厅：建议拍整体环境。
2. 沙发：建议拍沙发表面。
3. 卧室：重点拍地毯情况。
4. 厨房：建议拍整体。

操作步骤：

1. 展开“房间检查照片”。
2. 每个区域点击“添加”拍照。
3. 确认每个区域状态从“未拍”变成“可提交”。
4. 如需补充，可以继续添加照片。
5. 拍错的照片可长按删除。

注意：

1. 卧室优先拍地毯，因为这是检查重点。
2. 照片要能说明房间状态，不要只拍局部或模糊画面。
3. 早入住任务不可跳过检查照片。
4. 只有在普通入住且客人已到、着急入住时，页面才可能允许“客人已到，跳过检查照片”；跳过前仍需完成消耗品确认和客人需求确认。

## 8. 客人特殊需求确认

如果任务卡或检查页面显示客人需求，检查完成前必须确认。

操作步骤：

1. 阅读客人需求内容。
2. 现场确认需求已完成。
3. 勾选“我已完成客人需求”。
4. 如果无法完成，先不要标记完成，提交问题反馈并通知负责人。

## 9. 标记已完成

完成检查照片和消耗品确认后，点击“进入标记已完成”。

进入完成页后，系统会校验：

1. 关键区域照片是否齐全。
2. 消耗品确认是否完成。
3. 客人需求是否已确认。

如果页面提示未满足，返回“检查与补充”补齐。

最后一步是“密码盒 / 给钥匙视频”：

1. 查看页面显示的旧密码和新密码。
2. 先按要求修改密码盒密码。
3. 如果是直接把钥匙给客人，也要拍视频留存。
4. 点击“拍视频并完成”。
5. 视频上传成功后，任务会自动完成。

注意：

1. 视频最长按 App 限制拍摄，确保画面能看清密码盒或给钥匙动作。
2. 如果新密码未填写或不确定，不要自行改密码，先联系负责人。
3. 上传中不要退出页面，等系统提示成功。

## 10. 检查员日终交接

纯检查人员的日终交接重点不是备用钥匙和脏床品，而是：

1. 剩余消耗品照片
2. Reject 床品登记

操作步骤：

1. 回到任务页底部。
2. 找到“日终交接”。
3. 点击“上传剩余消耗品”。
4. 至少上传 1 张当天检查结束后自己剩余消耗品的照片。
5. 如果有 Reject 床品，点击“新增 Reject”。
6. 填写床品类型、使用房号、数量。
7. 上传不合格床品退回照片。
8. 确认所有照片显示已上传后，点击“提交日终交接”。

如果当天没有 Reject 床品，可以不新增 Reject。

## 11. 检查人员常见错误

避免以下操作：

1. 只看房号，不看入住时间和高优先标签。
2. 没确认消耗品就直接标记完成。
3. 房间检查照片缺少某个区域。
4. 卧室只拍床，不拍地毯。
5. 发现清洁问题但只口头说，没有在 App 记录。
6. 把维修、损坏、遗留物误写成清洁问题。
7. 密码盒视频没拍清楚就提交。
8. 日终交接忘记上传剩余消耗品。

## 12. 每天推荐流程

1. 前一天晚上或当天开始前：排序检查任务。
2. 到达房源前：确认房号、地址、入住时间、客人需求。
3. 现场检查：先处理消耗品，再检查清洁和房源问题。
4. 拍检查照片：客厅、沙发、卧室、厨房每类至少 1 张。
5. 完成前：确认客人需求、修改密码盒、拍视频。
6. 全部任务后：提交检查员日终交接。

## 13. 完成标准

检查人员一天结束前应确认：

1. 自己负责的检查任务都已处理。
2. 消耗品确认或补充记录已提交。
3. 必要的清洁问题、房源问题已在 App 里反馈。
4. 检查照片已按区域补齐。
5. 密码盒 / 给钥匙视频已上传。
6. 日终交接已提交。
