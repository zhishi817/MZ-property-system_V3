# 清洁总览 交付自测清单

## 环境
- 浏览器：Chrome / Edge / Firefox 最新正式版
- 分辨率：1920×1080、1366×768
- 缩放：100%

## 布局与滚动
- 1920×1080 下无横向滚动条
- 1366×768 下无横向滚动条
- 关键模块无元素错位、无文本溢出遮挡

## 可访问性（WCAG 2.1 AA）
- 文本与背景对比度满足 AA（普通文本≥4.5:1，大号文本≥3:1）
- 可交互控件可通过键盘 Tab 聚焦，并有清晰焦点样式
- 图表与表格区域具备可识别的 aria-label/语义结构

## 视觉一致性
- 字体/字号/行高：与原稿一致
- 色值：与原稿一致
- 间距/圆角/阴影：与原稿一致
- 图标尺寸与位置：与原稿一致

## 还原度对照
- 已生成差异热图报告（像素差异占比 ≤ 0.5%）
- 基准图来自原稿导出的 PNG（同分辨率、同缩放）
